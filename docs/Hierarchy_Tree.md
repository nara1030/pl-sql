계층 구조 쿼리
=====
계층 구조는 하이라키(Hierarchy) 혹은 트리(Tree) 구조라고 불린다.

## 상황
* `계층 구조 쿼리` 사용 가능한 상황(아래 각 두 상황이 OLTP와 OLAP에 따라 나뉜 상황인지 **확인**)
	* 한 테이블에 UPPER 컬럼(예를 들어 empno의 UPPER 컬럼은 mgrno)이 있을 때
	* Snowflake 모델
		* 데이터 마트(데이터 웨어하우스?) 구성 시 차원 테이블(Dim) 관련 설계
* 흔히 접할 수 있는 예(요구상황)
	* 년-월-일
	* 지역-국가-시/도-군/구
